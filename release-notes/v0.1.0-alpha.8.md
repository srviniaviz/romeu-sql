# Romeu SQL 0.1.0-alpha.8

Correção focada no gerenciamento de usuários do cluster e execução de comandos administrativos.

## Destaques

- O gerenciador de cluster agora consegue executar fluxos administrativos com múltiplos comandos SQL em sequência.
- A criação de usuários no PostgreSQL passa a aplicar `CREATE ROLE`, `GRANT` e `ALTER DEFAULT PRIVILEGES` sem cair no erro de prepared statement com múltiplos comandos.

## Correções

- Corrigido o erro `cannot insert multiple commands into a prepared statement` ao criar usuários com presets de permissões.
- O executor administrativo agora divide statements por `;` respeitando strings, identificadores com aspas e comentários SQL.

## Aviso de alpha

Romeu SQL ainda está em desenvolvimento. Enquanto estiver em alpha, valide fluxos críticos antes de usar em dados sensíveis.

## Instalação

Baixe o instalador nos assets desta release.
