# Romeu SQL v0.1.0-alpha.14

Correcao para comparacao de versoes no verificador de atualizacoes.

## Destaques

- O app agora entende que `0.1.0-14` no instalador MSI e `0.1.0-alpha.14` no GitHub representam a mesma release.
- A janela de atualizacao nao deve mais aparecer quando a versao instalada ja corresponde a ultima release alpha.
- A versao exibida no modal foi normalizada para o formato alpha usado no GitHub.

## Correcoes

- Corrigida comparacao entre versoes numericas de pre-release usadas pelo MSI e tags `alpha` usadas no repositorio.
- Adicionados testes para impedir regressao nesse fluxo.

## Aviso de alpha

Romeu SQL ainda esta em desenvolvimento. Enquanto estiver em alpha, priorize validar fluxos criticos antes de usar em dados sensiveis.

Baixe o instalador nos assets desta release.
