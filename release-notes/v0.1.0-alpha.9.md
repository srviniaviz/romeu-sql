# Romeu SQL 0.1.0-alpha.9

Correção focada na edição inline de valores em linhas.

## Destaques

- A edição de valores no modal de linha ficou mais estável para alterações no meio de strings.

## Correções

- Corrigido o cursor pulando para o início do campo ao digitar em valores editáveis.
- O editor JSON-like agora evita sincronizar o conteúdo enquanto o campo está focado, preservando a posição do cursor durante a digitação.

## Aviso de alpha

Romeu SQL ainda está em desenvolvimento. Enquanto estiver em alpha, valide fluxos críticos antes de usar em dados sensíveis.

## Instalação

Baixe o instalador nos assets desta release.
