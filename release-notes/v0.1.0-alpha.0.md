# Romeu SQL 0.1.0-alpha.0

Primeira versão alpha pública do Romeu SQL para Windows.

## Destaques

- Aplicativo desktop local em Tauri para gerenciar conexões PostgreSQL.
- Navegação por bancos, tabelas, colunas, índices e linhas.
- Leitura paginada de dados com modos de visualização em lista, JSON e tabela.
- Editor SQL com autocomplete e histórico local.
- Exportação de dados e resultados em CSV, JSON e Excel.
- Gerenciamento inicial de usuários, permissões e informações do cluster.
- Benchmark analyse para inspecionar tabelas e sinais básicos de performance.
- Configurações locais para Query, Editor SQL, Data View e Security.
- Tema claro/escuro e interface traduzida em PT-BR e EN.

## Aviso de alpha

Esta versão ainda está em desenvolvimento. O foco atual é PostgreSQL no Windows.
Outras engines, automações de atualização e mais opções avançadas ainda estão planejadas.

## Instalação

Baixe o instalador nos assets desta release. Se não houver asset anexado, a build ainda não foi publicada corretamente.
