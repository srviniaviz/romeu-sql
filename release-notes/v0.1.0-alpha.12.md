# Romeu SQL v0.1.0-alpha.12

Correcoes para edicao de tipos PostgreSQL e fluxo de atualizacao do app.

## Destaques

- O check de atualizacao agora exibe um modal antes de baixar e abrir o instalador.
- O auto-check respeita a configuracao local e usa o mesmo modal quando encontra uma versao nova.
- Edicao de valores tipados no PostgreSQL agora aplica casts seguros para timestamps, datas, JSON, bytea, UUID e enums.

## Correcoes

- Corrigido erro ao editar colunas `timestamp without time zone` que chegavam ao banco como `text`.
- Mantido bind parametrizado no update, usando metadados de tipo apenas para montar casts no SQL.
- Atualizadas traducoes do modal de atualizacao em PT-BR e ingles.

## Aviso de alpha

Romeu SQL ainda esta em desenvolvimento. Enquanto estiver em alpha, priorize validar fluxos criticos antes de usar em dados sensiveis.

## Instalacao

Baixe o instalador nos assets desta release.
