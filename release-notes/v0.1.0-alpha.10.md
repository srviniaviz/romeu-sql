# Romeu SQL v0.1.0-alpha.10

Correcoes focadas na edicao de linhas, persistencia visual das acoes e SEO da landing.

## Destaques

- Modais de criar e editar linha agora permanecem abertos quando o salvamento falha.
- Acoes de inserir, atualizar e falhar agora exibem toast de feedback.
- Landing recebeu SEO completo para GitHub Pages, Open Graph, Twitter cards, canonical e JSON-LD.

## Correcoes

- Ajustado update de linhas no PostgreSQL para evitar comparacao invalida entre timestamp e texto.
- Melhorado editor de valores longos em linhas, usando area de texto onde faz sentido.
- Corrigidas traducoes de feedback para PT-BR e ingles.

## Aviso de alpha

Romeu SQL ainda esta em desenvolvimento. Enquanto estiver em alpha, priorize validar fluxos criticos antes de usar em dados sensiveis.
